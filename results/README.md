# results

このディレクトリは、vega investigations の実験結果を
「再実行できる形」で清書・集約するための場所です。

## 定義

このノート群でいう「実験」は、次を含む一連の作業を指します。

- コードトレース（静的調査）
- 実行ログ取得（動的調査）
- 逆アセンブル確認（必要な場合）
- 上記を根拠にした結論化

## 記録ポリシー

- 推測と事実を分ける。
- 事実には根拠（ログ、コード位置、コマンド）を添える。
- 1ファイル1テーマで記録する。
- 結論だけでなく「再現手順」を残す。

## ディレクトリ構成

- `common`: GPU 非依存の知見（ROCm 全体、設計、共通 fallback など）
- `MI25`: MI25 実機での観測結果
- `Vega56`: Vega56 実機での観測結果
- `Vega64`: Vega64 実機での観測結果

## ファイル命名ルール（推奨）

- `YYYYMMDD_<topic>.md`
- 例: `20260313_miopen_solver_selection.md`

## 1本ごとの最小テンプレート

以下の見出しを最低限そろえる。

1. 目的
2. 環境
3. 実行コマンド
4. 観測結果
5. 根拠リンク（ログ/コード）
6. 判定（confirmed / not_confirmed / need_more_cases）
7. 次アクション
